General

á      Be accurate with your descriptions

o   E.g. graphs goes from -10 to 10

o   Instead write goes from -10 to 10 in increments of 1

á      Write about all important parts of the graphic

o   E.g. The ledgened of curve important but colour is not

á      Only describe what is on the graphic

o   Title

o   Axis

o   Curve

o   Max and min

o   Labels

á      Only talk about necessary information

o   Eg. The colour of a graph is not necessary (Unless specifically asked).

á      Use common language to explain

o   Eg. Rather than talking about how it looks like a Ôsine graphÕ talk about how it looks like a wave. The recipient may not know what a sine graph looks like!

á      Use appropriate language from STEM discipline tailored for the recipient

o   E.g. use phrase ÒcurveÓ not ÒlineÓ.

á      If more than one feature, identify each and start describing attributes corresponding to one feature, then move to the second feature.

o   E.g. The image has a red, equilateral triangle drawn using a solid line, and a green circle plotted with dashed line inside the triangle.

á      Be clear and concise with your answer

á      Be polite!

o   E.g. Don't forget to say Hi, and an engaging sentence to say Bye. There is a computer interface between you and an other IRIS member, but we want to sustain a friendly, supportive community

Line Graph

1.    Brief summary including axis titles and ranges

2.    Include style features, number and position of intersections.

3.    Describe minima, maxima and any extreme point on a data feature.

4.    Comparison of data if more than one line

Example: Delta Function

Bad Description

The image is of a line graph. The blue line is constantly at zero until it peaks to 1 at x = 0.

 

The title and axis have not been described. The colour of the line is also not physically significant and so can be left out.

 

Good Description

The line graph is titled ÒDelta FunctionÓ. The x-axis, showing position, has a range -50 to 50. The y-axis showing the value of delta function at the position x has a range from 0 to 1. The line is horizontal l at delta=0 until it sharply peaks at x = 0 at a value of delta=1. It then dips back to delta=0 for the rest of the graph.

Pie Charts

  1. Brief summary including title
  2. Data should be put into a table
  3. Data should be put from largest to smallest

Example

Bad Description

A chart showing the gas compositions in the atmosphere. Nitrogen (78%) in green, Oxygen (21%) in blue, and other gases (1%) in red.

 

Whilst the description does present the correct information to the user, it is jumbled with a lot of arbitrary details.

 

Good Description

A pie chart captioned ÒThe percentages of each of the gases in the atmosphere.Ó

Tables in descending order are especially useful when there is a large amount of data being presented.

á      Describe image from left to right

Good Description
